# Análisis de costos en centros de salud

Proyecto de análisis de datos usando SQL para identificar patrones de gasto en centros de salud, especialidades y pacientes.

## Objetivo
- Analizar gasto total por centro
- Identificar especialidades más costosas
- Detectar pacientes de alto consumo
- Incluir centros sin atenciones

## Herramientas
- MySQL

## Principales conclusiones
- El gasto se concentra en pocas especialidades
- Algunos centros presentan mayor gasto que el promedio
- Existen centros sin atenciones registradas
